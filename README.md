# Placement & Internship Board

A Vite + React application displaying recruitment and internship offers parsed for the 2027 batch of MIT Manipal.

## Development Setup

To run the application locally:

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Production Build

To build the application for deployment:

```bash
# Build the project
npm run build

# Preview build locally
npm run preview
```

## Vercel Deployment

This project is fully configured for Vercel deployment. You can import this repository directly in Vercel, and it will deploy with zero additional configuration using Vite presets.
